# Mars-YOLO-15km: Geomorphometric-Consistent Crater Rim Delineation

[![DOI](https://zenodo.org/badge/1212555189.svg)](https://doi.org/10.5281/zenodo.19610406)
[![Python 3.12+](https://img.shields.io/badge/python-3.12+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains the official codebase, benchmark datasets, pre-trained models, and validation frameworks for the paper: **Geomorphometric-Consistent Crater Rim Delineation from Orbital Data: Multi-Modal Validation, Radial Profiling, and Benchmarking Against Global Catalogs**.

## 📖 Overview
Existing automated Crater Detection Algorithms (CDAs) predominantly prioritize bounding-box classification over geomorphometric precision. This framework bridges the critical gap between computer vision localization and rigorous planetary geology.

By coupling the YOLOv8m architecture with a 30% Slicing Aided Hyper Inference (SAHI) tessellation strategy and a strict 15 km Minimum Resolvable Diameter (MRD), this model natively extracts the **true outer geomorphic rim** of Martian impact structures. The pipeline corrects the systematic "inward-rim bias" prevalent in historical manual mapping and is physically validated using Pixel-Intensity-to-Elevation (PITE) profiles, Explainable AI (XAI), and cross-sensor (THEMIS, TES, MOLA) alignment.

## 🚀 Quickstart & Installation

**1. Clone the repository**

```bash
git clone [https://github.com/YourUsername/Mars-YOLO-15km-Framework.git](https://github.com/YourUsername/Mars-YOLO-15km-Framework.git)
cd Mars-YOLO-15km-Framework
```

**2. Install dependencies**
Ensure you are using Python 3.12+. We provide a strictly pinned `requirements.txt` to guarantee reproducibility.

```bash
# Install the base requirements
pip install -r requirements.txt

# (Optional but Recommended) Force CUDA installation for GPU acceleration
pip install torch==2.2.2 torchvision==0.17.2 --index-url [https://download.pytorch.org/whl/cu118](https://download.pytorch.org/whl/cu118)
```

**3. Run the Interactive Tutorial**
We highly recommend starting with our interactive Jupyter Notebook. It automatically loads the pre-trained weights, runs inference on a sample THEMIS image, and immediately generates photogrammetric proof via an Explainable AI heatmap.

```bash
jupyter notebook notebooks/01_Interactive_Inference_and_XAI.ipynb
```

## 🔬 Reproducing Paper Figures

To reproduce the high-resolution (350 DPI), publication-grade figures presented in the manuscript, execute the following scripts from the root directory:

**1. Generate Formal Validation Metrics (Precision-Recall, F1, Confusion Matrix)**

```bash
python scripts/evaluate_metrics.py
```

*Outputs are saved to `runs/detect/val/`.*

**2. Generate Crater Size-Frequency Distributions (CSFD)**

```bash
python scripts/plot_csfd.py
```

*Outputs are saved to `validation_metrics/`.*

**3. Extract Explainable AI (XAI) Saliency Maps**

```bash
python scripts/generate_xai_maps.py
```

*Outputs are saved to `validation_metrics/XAI_Heatmaps/`.*

## 📊 Data Access & Scientific Catalogs

### 1\. The Refined Crater Catalog (For Planetary Geologists)

If you require highly precise, mathematically centered coordinate bounding geometries for regional CSFD analysis or surface age estimation, you do not need to run the neural network.

  * **Access:** Located locally in `catalogs/Refined_Mars_Crater_Catalog.csv`.
  * **Details:** Contains 2,598 refined bounding coordinates across Arabia, Elysium, Melas, and Noachis, including 12 uncatalogued high-confidence macro-basin discoveries.

### 2\. The Mars-YOLO-15km Benchmark Dataset (For Computer Scientists)

To accelerate next-generation planetary computer vision research, the complete 736-tile benchmark dataset is hosted directly in this repository within the `data/` directory.

  * **Official Academic Record:** For formal citation and long-term archival access, the dataset is also permanently hosted on Zenodo: [](https://www.google.com/url?sa=E&source=gmail&q=https://doi.org/10.5281/zenodo.XXXXXXX)
